Here is the 3-Step "Magic Link" Flow that makes setup instant for a non-tech client.
The "Gemin-Eye" Client Experience
Step 1: You send them a Telegram Link.
 * You send: t.me/GeminEyeBot?start=setup
 * They click it.
Step 2: The Bot interviews them (The Wizard).
 * Bot: "Welcome! Let's set up your spy. What is the name of your business?"
 * Client: "Mario's Tacos"
 * Bot: "Great. What are 3 keywords I should watch for? (e.g. tacos, hungry, mexican)"
 * Client: "tacos, burrito, lunch"
 * Bot: "Done. I'm now configured for Mario's Tacos."
Step 3: The "Magic Button" (The Bookmarklet).
 * Bot: "To start scanning Facebook, just drag this blue button to your browser's bookmarks bar."
 * (The bot sends a link that says [ üëÅÔ∏è Scan Group ]).
 * Client: Drags it to their bar. Done.
Why this works:
 * No Passwords: They stay logged into Facebook as themselves.
 * No Extensions: Extensions require review and store updates. A "Bookmarklet" is just a smart link.
 * Compliance: The script only runs when they click the button on a group page. It is 100% "User-Initiated."

The "Spy Glass" Script (The Secret Weapon)
You noticed the wizard references https://gemin-eye-kmages.replit.app/public/spy-glass.js. We need to create that file in Replit so the bookmarklet can load it.
Create a new file in Replit called public/spy-glass.js:
/* üëÅÔ∏è THE SPY GLASS (Client Side) */
(function() {
  console.log("üëÅÔ∏è Gemin-Eye Activated");
  
  // Visual Feedback for the Client
  const banner = document.createElement("div");
  banner.style = "position:fixed;top:0;left:0;width:100%;background:#0088cc;color:white;text-align:center;padding:10px;z-index:9999;font-weight:bold;";
  banner.innerText = "üëÅÔ∏è GEMIN-EYE IS WATCHING THIS FEED...";
  document.body.appendChild(banner);

  let seenPosts = new Set();

  function scan() {
    // Find all posts on Facebook (Class names change, but 'dir="auto"' usually catches text)
    const posts = document.querySelectorAll('div[dir="auto"]');
    
    posts.forEach(post => {
      const text = post.innerText;
      if (text.length > 20 && !seenPosts.has(text)) {
        seenPosts.add(text);
        
        // Send to your Replit Bot
        fetch('https://gemin-eye-kmages.replit.app/api/telegram/webhook', {
          method: 'POST',
          headers: {'Content-Type': 'application/json'},
          body: JSON.stringify({
            message: {
              text: "FB_SCAN: " + text,
              chat: { id: "YOUR_TELEGRAM_ID_HERE" } // We will make this dynamic later
            }
          })
        });
        
        // Highlight the post so the client knows it was sent
        post.style.border = "3px solid #0088cc";
      }
    });
  }

  // Scan every 2 seconds as they scroll
  setInterval(scan, 2000);
})();
